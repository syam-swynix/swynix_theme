# Swynix Theme

Custom theme application for SwynixERP built on Frappe Framework.

## Features
- Custom branding and color schemes
- Responsive design components
- SwynixERP specific UI/UX enhancements

## Installation

### Step 1: Get the app
```bash
bench get-app https://github.com/syam-swynix/swynix_theme.git
```

### Step 2: Install on site
```bash
bench --site [your-site-name] install-app swynix_theme
```

## Version
- **Current Version**: 1.0.0
- **Compatible with**: Frappe v15.x, ERPNext v15.x

## Dependencies
- Frappe Framework
